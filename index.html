HIGH AND LOW

ルール説明

伏せてあるカードがもう一方のカードより、大きいか小さいかを当てるゲームです。

  • カードの強さは以下の通りです。
    (弱)A < 2 < 3 < 4 .... J < Q < K(強)
  • 数字が同じ場合はマークで判定します。強さは以下の通りです。
    (弱)♣ < ♦ < ♥ < ♠(強)
  • すべてのカードを使い終えた時に評価がでます（x~y% は「x%未満y%以上」を示す）。
    • 100% : PERFECT
    • 100～60% : EXCELLENT
    • 60～40% : OK
    • 40～0%: NOOB
ターン デッキ残数（枚） 勝ち（回） 負け（回） 勝率（％）
